CSS Position Property has 4 possible values

  1. Static
  2. Relative
  3. Fixed
  4. Absolute
  5. Sticky

Static

Staticis the default position value

Top, bottom, left and right will not work with static.

A static element will show up exactly where it is in the HTML, left to right, top to bottom... it CANNOT be moved

Static Box

Relative Position

A relative position is teh same as static, EXCEPT it will honor top, left, bottom, right

If you move it, it will sit on top of other stuff on teh page, and the page will think its still where it used to be.

Relative Box

Fixed position

A fixed position elements will remain, relative to teh viewport(the screen) even if the user scrolls

Top, bottom, left, and right will work just fine

Unlike relative, it will not reserve its old space. I.e., anything with static or relative will not know it's there

Fixed Box

Absolute position

Absolute elements work off the grid system. Just like fixed, they are ignored by EVERYONE.

They honor z-index

Like fixed, they do not leave gaps in the page

the top/left/bottom/right will be relative to the first parent that has a position set

Absolute Box

sticky position

Sticky elements are just like fixed, except when the element would reach teh top they will vanish

Sticky Box
